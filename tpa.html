T.P.A. - Técnica de Programação e Algoritmos

Sobre a matéria de T.P.A.

Esta matéria visa desenvolver os conseitos básicos da programação, como variaveis, tipos de dados e estruturas condicionais, ensinando o aluno a pensar de forma lógica utilizando e criando algoritmos para resolução de problemas da programação.

Atividades realizadas em sala de Aula

Desconto

Este Algoritmo lê um valor do usuário e retorna o valor com um desconto de 10%.

Flappy Bird

Projeto durante as aulas da matéria de APS, com todo o acompanhamento do professor Lúcio, sendo um pássaro que pula entre os troncos, também com pontuação, feito somente com HTML, CSS e JavaScript.
